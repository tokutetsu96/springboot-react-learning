# アーキテクチャ概要

## システム構成図

```text
┌─────────────────────────────────────────────────────┐
│                    Docker Network                    │
│                                                     │
│  ┌──────────────┐    ┌──────────────┐    ┌────────┐ │
│  │   Frontend   │    │   Backend    │    │  DB    │ │
│  │  React/Vite  │───▶│ Spring Boot  │───▶│Postgres│ │
│  │  :5173       │    │  :8080       │    │ :5432  │ │
│  └──────────────┘    └──────────────┘    └────────┘ │
│         ▲                                           │
└─────────┼───────────────────────────────────────────┘
          │ ブラウザ
     localhost:5173
```

## リクエストフロー

```text
ブラウザ
  │
  ▼
React (Vite dev server :5173)
  │ /api/* へのリクエストをプロキシ
  ▼
Spring Boot (:8080)
  │ Controller → Service → Repository
  ▼
PostgreSQL (:5432)
```

## レイヤードアーキテクチャ

```text
Controller      HTTPリクエスト/レスポンスの処理
    │               ・ルーティング
    │               ・入力バリデーション (@Valid)
    │               ・HTTPステータスコードの決定
    ▼
Service         ビジネスロジック
    │               ・ユースケースの実装
    │               ・トランザクション管理
    │               ・ドメインルールの適用
    ▼
Repository      データアクセス
                    ・JpaRepository によるCRUD
                    ・カスタムクエリ
```
